# PSB Mrówka Bysewo - Multi-store Retail Platform

A modern, responsive web application developed for a regional cluster of **PSB Mrówka** DIY stores (Chojnice, Tuchola, Człuchów, Borkowo). This project serves as a digital catalog and customer service hub, featuring real-time inventory checks and lead generation tools.

## 🔗 Live Project
View the live site here: **[mrowkibysewo.pl](http://mrowkibysewo.pl)**

## 🚀 Key Technical Features
* **Dynamic Inventory System:** Implemented a JSON-based database logic to handle product listings and real-time stock availability across 4 different warehouse locations.
* **Advanced Search Engine:** Custom JavaScript search algorithm supporting instant filtering by product names, EAN codes, and categories.
* **Interactive UI Components:** * Dynamic Product Modals with encoded URL descriptions.
    * Lead generation system for custom window pricing (Drutex integration).
    * Automated promotional sliders and seasonal sale filters.
* **Responsive Web Design (RWD):** Built with a mobile-first approach using **Tailwind CSS**.

## 🛠️ Tech Stack
* **Language:** HTML5, Modern JavaScript (ES6+)
* **Styling:** Tailwind CSS
* **Backend Integration:** Formspree API for quote requests.
